# MoneyPrinterFaster

> Type one sentence, AI auto-generates short videos. From script to final cut, zero manual intervention.

**Languages:** [English](README.en.md) | [简体中文](README.md) | [繁體中文](README.zh-TW.md) | [Español](README.es.md) | [العربية](README.ar.md) | [Français](README.fr.md)

An automated short video production pipeline built with Go. Just provide a topic — the system automatically handles LLM script generation → TTS voiceover → subtitle generation → material download → audio-video composition, ultimately outputting an MP4 video with subtitles + BGM.

**Live Demo: [http://47.99.221.247:8080/](http://47.99.221.247:8080/)**

## Quick Start

### Prerequisites

- Go 1.22+
- FFmpeg (with libx264 support)
- Python 3 + edge-tts (`pip install edge-tts`, auto-installed on first run)

### Configuration

```bash
cp config.example.toml config.toml
```

Edit `config.toml` and fill in API Keys:

| Config | Description | Required |
|--------|-------------|----------|
| `[llm.deepseek].api_key` | DeepSeek API Key | Yes (at least one LLM) |
| `[material.pexels].api_keys` | Pexels API Key list (video material search) | Yes (at least one material source) |
| `[material.aliyun_image].api_key` | Alibaba Cloud Bailian API Key (AI text-to-image) | No (choose one) |
| `[tts]` | TTS engine (default: edge, no key needed) | No |

### Run

```bash
# Development mode
make run

# Build
make build
./build/moneyprinterFaster

# Cross-compile
make build-darwin
make build-linux
```

Visit `http://localhost:8080` after startup.

### Workflow

1. Enter a video topic (e.g., "The future of artificial intelligence")
2. Click "AI Generate Script" to auto-generate, or write manually
3. Choose material source: **Video Search** (Pexels) or **AI Image Generation** (Z-Image)
4. Adjust parameters (language, video aspect, voice, etc.)
5. Click "Submit Task"
6. Monitor progress in the task list, download MP4 when complete

### Material Sources

| Source | Description | Cost |
|--------|-------------|------|
| **Video Search (Pexels)** | Auto-search royalty-free video clips by keywords | Free |
| **AI Image Generation (Z-Image)** | Alibaba Cloud Bailian text-to-image, generates AI images with Ken Burns zoom effect | $0.015/image |

## Core Features

- **Multiple Material Sources**: Pexels video search / Alibaba Cloud Bailian Z-Image text-to-image, selectable per task
- **AI Image-to-Video**: Z-Image generated photos auto-converted to video clips with Ken Burns zoom effect
- **Real-time Cost Estimation**: When AI image generation is selected, the frontend estimates image count and cost based on the script
- **CPU-Adaptive Worker Pool**: Auto scales based on CPU load, prevents overloading
- **SQLite Persistent Queue**: Tasks survive service restarts, supports historical task queries
- **FFmpeg Memory Optimization**: Per-file normalization + concat demuxer streaming, avoids OOM
- **Dual Subtitle Strategy**: Prefers libass subtitles filter, falls back to drawtext
- **Precise Duration Control**: ffprobe detects real duration, material download stays within 110% of audio duration
